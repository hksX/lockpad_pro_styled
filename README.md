## LockPad Pro — Styled Edition (Offline assets included) ##

## After unzip: ##
open terminal in  vs code
1. npm install @vitejs/plugin-react --save-dev
2. cd lockpad_pro_styled
3. npm install
4. npm run dev
5. Open the URL printed by Vite (usually http://localhost:5173)
